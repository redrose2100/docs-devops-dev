安装mkdocs

pip install mkdocs

创建一个项目

mkdocs new MyMkdocsDemo

启动项目

cd MyMkdocsDemo
mkdocs serve

查看项目运行页面

浏览器打开 http://127.0.0.1:8000/ 即可

部署github page

mkdocs gh-deploy
